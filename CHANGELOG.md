# Changelog

## 2026-03-19

### Added
- 地图视角系统：支持“跟随/自由观察”双模式、鼠标拖拽平移、回到化身按钮、视角坐标与模式显示。
- 目标可达性提示：状态栏与地块详情增加“可达/不可达”信息，自动探索目标筛选对齐当日半径规则。
- 资源面板重构：新增 `库存/背包/装备` 三个子Tab，未拥有物品默认不显示。
- 统一 SVG 物品图标体系：资源栏、菜单、图鉴、商店、强化、开箱弹窗统一风格图标。
- 物品交互增强：数量选择（减/加/输入/全部）、批量使用/回收、物品词典快捷打开。
- 开箱结算弹窗：支持批量开箱，展示本次奖励并支持悬浮查看物品详情。
- 装备槽位系统：槽位化装备栏、点击槽位可装备/替换/移除、装备总加成明细面板。
- 调试战斗增强：一键触发战斗支持选择普通/精英/BOSS/当前位置怪物。
- 世界事件（基础版）：灵潮涌动/兽潮躁动/云集坊市，带持续时间与实时效果。
- 新增“回血丹”：可合成、可商店购买、可携带并在战斗中自动补血。

### Changed
- 主角境界改为纯等级制显示：`Lv.1 ~ Lv.10`，并在修炼面板与状态面板展示当前修为和距离下一级所需修为。
- 去除“突破/自动突破”机制，改为修为达标后自动升阶。
- 战斗面板新增敌方战斗属性展示（攻击/防御），精英与BOSS战显示高威胁标识。
- BOSS 刷新范围与可达规则对齐：超远 BOSS 不再刷为不可达目标（超范围时降级为精英）。
- 资源栏信息密度提升：物品格改为固定方形高密度布局，悬浮展示详情，交互高亮与禁文本选中。
- 物品菜单视觉重做：数量控件与按钮布局压缩，回收按钮实时显示“数量×单价=总金币”。
- 装备规则改为单槽唯一：不再叠装，替换时自动卸下旧装备回背包。
- 地图解读默认隐藏，支持按钮展开/收起。

### Fixed
- 修复“Day 1 循环返城”相关流程问题，自动探索与返城切换逻辑更稳定。
- 修复化身背包实时刷新与“采一次就回城”异常，补全采集效率展示。
- 修复建造按钮不可点击与建造材料显示（`当前/需求`，不足红色、满足绿色）。
- 修复存档字段兼容：新增视角、资源子Tab、装备槽位、世界事件等状态可正确保存/读取。
- 修复开箱弹窗层级：悬浮详情不再被弹窗遮挡。
- 修复装备槽位图标不显示与悬浮明细缺失问题。

## 2026-03-18

### Added
- 主角“肉身修炼”面板：圆形周期进度、开始打坐、突破、自动突破。
- 修炼系统合并：打坐与挂机结算统一进入同一修为池与突破流程。
- 丹药商店（金币购买）：聚灵丹/回春散/护体符/凝神符。
- 装备强化系统：最高 +10，每级提升装备加成 12%。
- 化身战斗属性面板：HP/MP/攻击/防御/命中/闪避/暴击实时展示。
- 新文档：`REALM_PROGRESSION.md`（境界经验设计详细方案）。

### Changed
- 战斗节奏下调：回合结算间隔加长，整体战斗时长增加。
- 怪物强度上调：整体血量提升。
- 精英与BOSS显著强化：天劫妖王与精英妖将的生命、攻防大幅提高。
- 掉落整体抬升：普通资源、稀有材料、宝箱与高阶掉落概率提升。
- 左侧UI重构：
  - 探索控制改为“化身面板”，仅保留自动探索与回城优先。
  - 功能区收敛，空间占用更紧凑。
- 文案统一：去除“采气”表述，统一为“打坐修炼”。

### Fixed
- 存档系统升级至 v2，并兼容迁移旧存档。
- 读档时库存容器先重置再覆盖，避免旧字段残留污染。
- 移除控件后补充DOM存在性判断，避免空引用事件绑定错误。
